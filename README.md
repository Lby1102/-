# 临时门户网站

通过 GitHub Pages 部署的临时门户网站

## 🚀 部署方式

本网站使用 GitHub Pages 进行部署，访问地址：
- **主要地址**: https://lby1102.github.io/-/

## 📁 项目结构

```
临时门户网站/
├── index.html          # 主页面
├── styles.css          # 样式文件
├── script.js           # JavaScript 脚本
└── README.md           # 项目说明
```

## 🔄 更新网站

### 方法 1: 通过命令行
```powershell
# 添加所有更改
git add .

# 提交更改
git commit -m "更新网站内容"

# 推送到 GitHub
git push origin main
```

### 方法 2: 通过 VS Code
1. 在源代码管理面板中查看更改
2. 输入提交信息
3. 点击"提交"按钮
4. 点击"同步更改"或"推送"

## ⚙️ 启用 GitHub Pages

1. 访问 GitHub 仓库: https://github.com/Lby1102/-
2. 进入 Settings (设置)
3. 在左侧菜单找到 "Pages"
4. 在 "Source" 下选择分支 (通常是 `main`)
5. 保存后等待几分钟，网站将自动部署

## 🛠️ 本地预览

在浏览器中直接打开 `index.html` 文件，或使用 VS Code 的 Live Server 扩展进行实时预览。

## 📝 修改内容

- 编辑 `index.html` 修改页面内容
- 编辑 `styles.css` 修改样式
- 编辑 `script.js` 添加交互功能
